# Asil Portfolio

Hi, I'm Asil Doğan Gümüş. This is my personal portfolio website, designed to showcase my projects, skills, and contact information as a young developer.

## Features

- Modern and fast full-stack application
- React frontend and FastAPI backend
- Display projects categorized by type
- Responsive and mobile-friendly design
- Uses MongoDB for data management
- Fast and secure API integration

## Technologies

- Frontend: **React**, Tailwind CSS
- Backend: **FastAPI**, Python
- Database: **MongoDB**
- Deployment: Netlify / Vercel / Heroku (your choice)
